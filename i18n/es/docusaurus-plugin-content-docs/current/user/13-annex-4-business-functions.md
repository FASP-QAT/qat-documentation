# Anexo 4: Funciones Empresariales

> Las funciones empresariales son un conjunto de capacidades disponibles para el usuario. Se utilizan al crear funciones (consulte **Funciones (administrador de aplicaciones** **solo)**). Cada rol tiene una o más funciones comerciales.
>
> Para datos de nivel maestro, los verbos utilizados en las funciones comerciales son los siguientes:

- **Lista**: permite al usuario ver la tabla con los datos existentes para ese elemento de datos maestros. Si solo se elige "Lista X", el usuario solo podrá ver "X", pero no podrá editar ni agregar.

- **Editar**: permite al usuario actualizar o editar el elemento de datos maestros. Nota: Incluya siempre "lista" junto con "editar" o el usuario no podrá ver la pantalla para editar elementos de datos. **(NOTA: Una entidad no se puede eliminar. Los usuarios solo pueden "Desactivar" entidades).**

- **Agregar**: permite al usuario agregar un nuevo registro para el elemento de datos maestros. Nota: Incluya siempre "lista" junto con "agregar" o el usuario no podrá ver la pantalla para agregar elementos de datos.

- **Administrar**: permite al usuario realizar todas las funciones antes mencionadas

La función empresarial anterior se puede proporcionar a los usuarios en cualquier combinación. Por ejemplo:

- El usuario A solo tiene acceso a "Lista", por lo que solo puede ver los datos existentes ("solo lectura")

- El usuario B solo tiene acceso "Lista" + "Editar", por lo que puede ver y editar datos existentes, pero no puede agregar entidades "nuevas".

- El usuario C tiene acceso "Lista" + "Agregar", por lo que puede agregar nuevas entidades y ver la lista de entidades.

- El usuario D tiene acceso “Agregar” + “Editar” + “Lista” (Administrar) que le permite realizar las 3 acciones.
