# Anexo 4: Funções Empresariais

> Funções de negócios são um conjunto de capacidades disponíveis para o usuário. Eles são usados ​​ao criar funções (consulte **Funções (administrador de aplicativo** **somente)**). Cada função tem uma ou mais funções de negócios.
>
> Para dados de nível mestre, os verbos usados nas funções de negócios são os seguintes:

- **Lista** – Permite ao usuário visualizar a tabela com os dados existentes para aquele elemento de dados mestre. Se apenas “Lista X” for escolhido, o usuário poderá ver apenas “X”, mas não poderá editar ou adicionar.

- **Editar** – Permite ao usuário atualizar ou editar o elemento de dados mestre. Nota: Sempre inclua “lista” junto com “editar” ou o usuário não poderá ver a tela para editar os elementos de dados. **(NOTA: Uma entidade não pode ser excluída. Os usuários só podem ‘Desativar” entidades.)**

- **Adicionar** – Permite ao usuário adicionar um novo registro para o elemento de dados mestre. Nota: Sempre inclua “lista” junto com “adicionar” ou o usuário não poderá ver a tela para adicionar elementos de dados.

- **Gerenciar**: permite ao usuário executar todas as funções mencionadas acima

A função comercial acima pode ser fornecida aos usuários em qualquer combinação. Por exemplo:

- O usuário A tem acesso apenas à “Lista”, portanto, só pode visualizar os dados existentes (“somente leitura”)

- O usuário B tem apenas acesso “Lista” + “Editar”, portanto pode visualizar e editar dados existentes, mas não pode adicionar ‘novas’ entidades.

- O usuário C possui acesso “Lista” + “Adicionar”, para que possa adicionar novas entidades e visualizar a lista de entidades.

- O usuário D possui acesso “Adicionar” + “Editar” + “Listar” (Gerenciar) que permite realizar todas as 3 ações.
