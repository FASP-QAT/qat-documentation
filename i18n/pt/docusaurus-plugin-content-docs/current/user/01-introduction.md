# Introdução

Bem-vindo à documentação do usuário da Quantification Analytics Tool (QAT). Este guia foi desenvolvido tanto para usuários finais quanto para administradores técnicos, fornecendo informações abrangentes sobre a funcionalidade do sistema, fluxos de trabalho e práticas recomendadas.

A Quantification Analytics Tool (QAT) é uma solução modernizada para previsão e planejamento de fornecimento liderados pelo país. QAT aproveita tecnologias avançadas para aprimorar a funcionalidade de ferramentas legadas como PipeLine e Quantimed.
