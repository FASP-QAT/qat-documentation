# Introdução

Bem-vindo ao Manual do Usuário da Quantification Analytics Tool (QAT). Este manual foi desenvolvido para usuários finais, fornecendo informações abrangentes sobre a funcionalidade do sistema, fluxos de trabalho e práticas recomendadas. Os administradores técnicos devem visualizar as seções [Desenvolvedor](/docs/developer/01_purpose.md) e [API](https://fasp-qat.github.io/qat-documentation/docs/api).


A Quantification Analytics Tool (QAT) é uma solution modernizada para previsão e planejamento de fornecimento liderados pelo país. A QAT aproveita tecnologias avançadas para aproveitar a funcionalidade de ferramentas legadas como PipeLine e Quantimed.
