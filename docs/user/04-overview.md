# Section 1. Overview

Welcome to the User Manual for the Quantification Analytics Tool (QAT)! This manual is designed for end-users (supply planners, forecasters, and viewers) and provides information on system functionality, workflows, and best practices. We hope you find this document useful. 

For an overview of the User Manual document structure, please refer to the [Table of Contents](03-table-of-contents.md).

For high-level information about QAT, please see the [About](https://fasp-qat.github.io/qat-documentation/docs/product/intro) section.

For technical documentation, please refer to the [Developer](/docs/developer/01_purpose.md) section.
